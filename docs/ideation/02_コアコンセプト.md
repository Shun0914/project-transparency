# コアコンセプト

## 哲学的な基盤

### タレブの洞察：「退職届を机の中に」

思想家ナシーム・タレブは、退職届を机の引き出しに忍ばせていたという。それがあるだけで、今日という日が「ご褒美」に感じたと語っている。

**「いつでも逃げられる」と分かると、プレッシャーから解放される**
- 今日という日が「義務」から「選択」に変わる
- 仕事が強制ではなく、自分の意志で選んだものになる
- 心理的な余裕が生まれる

### 逆説的な真実

**「いつでも辞められるからこそ、辞めなくなる」**

#### なぜこれが機能するのか

1. **心理的な余裕が生まれる**
   - 「逃げ道がある」と分かると、プレッシャーが減る
   - 追い詰められていないから、冷静に仕事できる
   - 結果的に辞める理由が減る

2. **上司もプレッシャーをかけなくなる**
   - 「この人、いつでも辞められる状態だな」→ 大事にしなきゃ
   - 無茶振りが減る、尊重されるようになる
   - **パワーバランスが健全化する**

3. **「辞める」が選択肢じゃなくなる**
   - 「辞めたいけど辞められない」→ ストレス溜まる → 爆発して辞める
   - 「いつでも辞められる」→ 辞める必要がない → 居続ける
   - **自由があるからこそ、自由に縛られなくなる**

### 従業員全員が「退職届を机の中に持つ」状態を作ったら？

#### 組織に起こる変化

1. **パワーバランスが健全化する**
   - 非対称な権力関係が崩れる
   - 上司と部下が対等な関係になる

2. **ストレスが減り、パフォーマンスが上がる**
   - 「辞められない」プレッシャーがなくなる
   - 逃げ道があるから、リスクを取れる
   - 心理的安全性が生まれ、創造性が高まる

3. **本当に辞めるべき人だけが辞める**
   - 感情的に辞めるケースが減る
   - 「もう無理！」ではなく、「次のステップに進もう」という冷静な判断
   - 円満な退職が増える

4. **属人化が減る**
   - みんなが「いつでも引き継げる状態」を維持
   - 情報がオープンになる
   - 組織の脆弱性が減る

5. **会社が社員を大事にせざるを得なくなる**
   - 「辞めさせない」ではなく、「辞めたくなくなる環境を作る」
   - ブラック企業は淘汰される
   - 市場原理が働く

## 「終わりから想像する」ことの重要性

退職という「終わり」を想像することで、今が輝く。

これを従業員全員が持つことで：
- 今日という日が「選択した結果」になる
- 仕事が「義務」から「ギフト」になる
- 組織が健全になる

## コアコンセプト：「%スコアリング」による可視化

### 本質的な問題の再定義

問題は「暗黙知を可視化する」ことではない。

**問題は「何が書かれていないか、誰も分かっていない」こと**

### ソリューション：ドキュメント化率のスコアリング

プロジェクトメンバー全員が、既存のドキュメント（Confluence等）を見て：
**「このプロジェクト、何%ドキュメント化されてると思う？」**

#### 例
- Aさん：「70%」
- Bさん：「60%」
- Cさん：「80%」
- Dさん：「50%」

→ **平均：65%**
→ **残り35%は何？**

### なぜこれが機能するのか

#### 1. 認識のズレが可視化される
- 「みんな分かってる」と思ってたのに、実は分かってなかった
- 数字があるから、ごまかせない
- Dさんだけ50%と言っている → 何が見えていないのか？

#### 2. 対話のトリガーになる
- 「なんで30%足りないと思うの？」
- 「私は20%と思ったけど、あなたは40%と言ってる。何が違う？」
- **議論が生まれる**

#### 3. 上司が問いを投げられる
- 「みんなが30%足りないと思ってる。残り30%は何だと思う？」
- この問いがあるから、チームで議論できる
- **定量があるから、対話が始まる**

#### 4. 主観でいい
- 正解はない
- みんなの主観の集合が、真実に近づく
- 完璧じゃなくていい

#### 5. シンプル
- %だけ
- 複雑な機能はいらない
- 誰でも使える

### AIが推測する必要はない

重要なのは「何が足りないか」を特定することではなく、**「足りていないことに気づく」こと**

- %だけで、気づきが生まれる
- 議論が価値を生む
- AIで自動化すると、対話が失われる

### 完璧を目指さなくていい

- 100%は無理
- でも80%なら現実的
- **「これで十分」が分かる**

- 全部書く必要はない
- 致命的な20%だけ埋めればいい
- **効率的**

## これは「引き継ぎツール」ではない

### 本質：プロジェクトの全体像を測る体温計

**日常的に使う価値**

#### 退職時だけじゃない
- 週次で「今週、%変わった？」
- 新しいメンバーが入った時「あなたは何%だと思う？」
- プロジェクトが複雑になった時「%下がってない？」

#### 組織の健全性の指標
- チーム全体の平均が70%を下回ったら危険信号
- メンバー間で認識のズレが大きい（標準偏差が大きい）→ コミュニケーション不足
- **組織のバイタルサイン**

### 応用シーン

#### オンボーディング
- 新入社員「私は今、30%しか理解してない」
- 1ヶ月後「50%になった」
- 3ヶ月後「70%になった」
- **成長が可視化される**

#### プロジェクトの引き継ぎ
- 前任者「私は90%理解してる」
- 後任者「私は今、20%」
- 1週間後「40%」
- 1ヶ月後「70%」
- **引き継ぎの進捗が見える**

#### チームの成熟度
- 立ち上げ直後：平均50%
- 半年後：平均75%
- 1年後：平均85%
- **チームの成長が測れる**

### ミッション

### ビジョン
「いつでも辞められる自由が、辞めたくない組織を作る。」

### バリュー
- 透明性が自由を生む
- 逃げ道があるから、挑戦できる
- 終わりを想像するから、今が輝く

## キャッチコピー案

1. **「いつでも辞められる、だから辞めない。」**
   - 逆説的で印象に残る
   - 哲学的で深い

2. **「自由な働き方は、透明性から始まる。」**
   - ポジティブ
   - 会社にも売りやすい

3. **「あなたの仕事、見える化していますか？」**
   - シンプル
   - 問いかけ形式で刺さる

4. **「明日、あなたが休んでも、仕事は回りますか？」**
   - リアル
   - 自分ごと化しやすい

5. **「何%、伝えられますか？」**
   - 直接的
   - スコアリングの本質を突く
